enum pparam::XObjectStatus::StatusTransition

Type of transitions between object statuses (e-g- from x to y).

Enumerator
DIRECT 

0 - There is a direct path from x to y. Transition could be done immediately.

D 

0 - Synonym for Direct.

WAIT 

1 - Object can wait, there is a chance that object changes his status from x to a status that has direct/none transition to target status(x).

W 

1- Synonim for INDIRECT

NONE 

2 - Don't wait, May be some changes in object status prepare situation for transition from x to y, but there isn't any guaranty for this changes to happen. So don't wait and cancle operation.

N 

Synonim for NONE.
