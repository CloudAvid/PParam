Detailed Description

template<typename T>
class pparam::XEnumParam< T >

Defines a parameter with enumeration values.

T would be a class with enumeration type empedded in. This class should have one specific value: "MAX": latest value in enum. Also this class should have "public static const typeString[MAX]" array that would store correspond string value for each enum value.
