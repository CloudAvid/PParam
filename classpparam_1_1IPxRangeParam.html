Detailed Description

Defines range of ip addresses. XML Format: <TAG-NAME>FROM[-TO]</TAG-NAME> TAG-NAME: Name of tag. Default is "ipx_range" FROM: IP address in form of IPv4 or IPv6 TO: Optional IP address in form of IPv4 or IPv6
