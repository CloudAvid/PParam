Detailed Description

template<typename Type>
class pparam::XObjectScanner< Type >

Objects from this class would used to scan connects "XObjects".

See "XObjects" connected area like of Graph or Tree, Scanner would scan graph in prefix order. Scanner when reaches any object, would call scan function and when return from any object to upper level, "scan_return" would called.
