Detailed Description

\breif Defines Port and Port Range

Author
Hamed Haji Hussaini(hajih.nosp@m.ussa.nosp@m.ini@c.nosp@m.loud.nosp@m.avid..nosp@m.com)

This class includes single port and port range at following formats: [!][[N]:]N OR [!]N[:[N]] Which N is a unsigned interger between 1 to 65536
