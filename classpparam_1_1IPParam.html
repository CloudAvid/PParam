Detailed Description

Peresent an IP address.

Author
Ali Esmaeilpour(esmae.nosp@m.ilpu.nosp@m.r@clo.nosp@m.udav.nosp@m.id.co.nosp@m.m)

This class represent IP (Internet Protocol) address. currently, its only a base for IPv4Param and IPv6Param classes.
