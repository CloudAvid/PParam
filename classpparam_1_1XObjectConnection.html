Detailed Description

template<typename Type>
class pparam::XObjectConnection< Type >

Defines a connection attributes between tow XObjects.

XObjects may have different connections with other XObjects in the repository. Each connection is bidirection, so when one object create a connection with other objects, that objects also would have connection with creator. Each XObject stores attributes of other side of connection.
